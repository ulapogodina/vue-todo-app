# Vue Todo App

Простое приложение "Список дел" на Vue 3 и TypeScript.

## Функциональность

- Отображение списка задач
- Карточка для каждой задачи
- Типизация с помощью TypeScript
- Стилизация с использованием Flexbox

## Технологии

- Vue 3
- TypeScript
- Vite
- Composition API

## Установка и запуск

1. Установите зависимости:
bash
npm install
